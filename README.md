# RandomizingGroupsSimulator
- This simulator randomizes groups of people based on how many groups the user wants. 
- Press the "Submit # of Groups" button to randomize groups
- After typing the name, you can press the "Enter" key also. 

# Updates
- You can separate 2 people from being in a group together, but you can also separate a different group of 2 people too, from being in a group together.
---Example: Separating Jack and Jill, separating Jack and Jake, and separating Jill and Molly is allowed
- Hover over a certain name in the names group to delete the name from the list of names.
- Hover over a certain group and delete it for that display of groups.
- You cannot add a name that is already in the list of names.
- Fixed multiple bugs with removing names and having groups with no people in them.
- Added a way for users to upload a list of names into the box of names.
- Fixed bugs with separating people in different groups, and having error messages if all the separation constraints cannot be fulfilled due to the number of groups and the number of people   requested.
