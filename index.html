How to Use the Randomizing Groups Simulator

Getting Started

  • Add Names: Type a name in the "Put in Names" field and click "Submit Name" or press Enter
  • Upload Names: Click "Upload Names" to upload a text file with names (one name per line)
  • Set Groups: Enter the number of groups you want in the "Groups #" field
  • Generate Groups: Click "Submit # of Groups" to randomize and create your groups

Advanced Features

  • Separate People: Use the dropdown menus to select two people who shouldn't be in the same group, then check the "Separate!" checkbox
  • Multiple Separations: Click "Add Another Separation" to create additional separation rules
  • Manage Separations: Use "Check All Separations" to enable all separation rules at once

Managing Your Data

  • Remove Names: Hover over any name in the Names box and click to remove it
  • Remove Groups: Hover over any group title in the Groups box and click to remove that group
  • Clear Everything: Click the "Clear" button to reset all data

Important Notes

  • You cannot add duplicate names to the list
  • Names cannot be blank
  • Separation constraints require at least 2 groups to work
  • The simulator will try to balance group sizes as evenly as possible
  • If separation constraints cannot be satisfied, you'll receive an error message with suggestions

Keyboard Shortcuts

  • Enter: Submit a name (when name input is focused)
  • Delete: Clear everything (same as Clear button)
